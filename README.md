### Updating Content
1. **Text Content**: Edit the HTML directly in `index.html`
2. **Colors**: Modify CSS custom properties in `:root` section of `styles.css`
3. **Images**: Replace placeholder icons with actual photos
4. **Contact Info**: Update phone, email, and address in the contact section

1. Add HTML structure following the existing pattern
2. Style with CSS using the design system
3. Add JavaScript functionality if needed
4. Update navigation menu
